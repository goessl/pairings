# pairings

A pairing functions Python package.
```python
>>> from pairings.nz import unfold
>>> for n in range(9):
...     print(n, unfold(n))
...
0 0
1 -1
2 1
3 -2
4 2
5 -3
6 3
7 -4
8 4
```

## Installation

```console
pip install git+https://github.com/goessl/pairings.git
```

## Usage

Enjoy the **[documentation webpage](https://goessl.github.io/pairings)**.
